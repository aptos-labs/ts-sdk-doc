  • Inherited from AptosClient.generateTransaction

    • Defined in Workspace/aptos-core/ecosystem/typescript/sdk/src/aptos_client.ts:250

    Generates a transaction request that can be submitted to produce a raw transaction that can be signed, which upon being signed can be submitted to the blockchain

    Parameters

    • sender: MaybeHexString

      Hex-encoded 16 bytes Aptos account address of transaction sender

    • payload: Types.TransactionPayload

      Transaction payload. It depends on transaction type you want to send

    • Optional options: Partial<UserTransactionRequest>

      Options allow to overwrite default transaction options. Defaults are:

        {
          sender: senderAddress.hex(),
          sequence_number: account.sequence_number,
          max_gas_amount: "1000",
          gas_unit_price: "1",
          gas_currency_code: "XUS",
          // Unix timestamp, in seconds + 10 seconds
          expiration_timestamp_secs: (Math.floor(Date.now() / 1000) + 10).toString(),
        }

    Returns Promise<UserTransactionRequest>

    A transaction object

transactionPending

  • transactionPending(txnHash: string): Promise<boolean>

  • Inherited from AptosClient.transactionPending

    • Defined in Workspace/aptos-core/ecosystem/typescript/sdk/src/aptos_client.ts:461

    Defines if specified transaction is currently in pending state

    Parameters

    • txnHash: string

      A hash of transaction

      To create a transaction hash:

      1. Create hash message bytes: "Aptos::Transaction" bytes + BCS bytes of Transaction.
      2. Apply hash algorithm SHA3-256 to the hash message bytes.
      3. Hex-encode the hash bytes with 0x prefix.

    Returns Promise<boolean>

    true if transaction is in pending state and false otherwise

waitForTransaction

  • Inherited from AptosClient.waitForTransaction

    • Defined in Workspace/aptos-core/ecosystem/typescript/sdk/src/aptos_client.ts:484

    Waits up to 10 seconds for a transaction to move past pending state

    example

    const res = await this.aptosClient.submitTransaction(signedTxn);

    // do smth after transaction is accepted into blockchain

    Parameters

    • txnHash: string

      A hash of transaction

    Returns Promise<void>

    A Promise, that will resolve if transaction is accepted to the blockchain, and reject if more then 10 seconds passed
